# go-payload-cryptography

## 1. Generate Private RSA Key of size 2048 using OpenSSL

```shell
$ openssl genpkey -aes-256-cbc -algorithm RSA -out private-key.pem -pkeyopt rsa_keygen_bits:2048
```

## 2. Extract the public key from the key pair

```shell
$ openssl rsa -in private-key.pem -outform PEM -pubout -out public-key.pem
```
